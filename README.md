# 局域网文件共享

可简单满足局域网内文件共享与消息发送

### 环境
    
    python 3.x

### 第三方依赖包

    # pip install web.py
    # pip install pillow
    # pip install qrcode

### 使用说明

> 运行工具

    # 默认使用 0.0.0.0:8080 地址
    # python run.py


> 浏览器中打开地址 127.0.0.1:8080
